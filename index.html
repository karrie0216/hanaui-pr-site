Message

広報は、担当者ひとりで
抱えなくて大丈夫です。

学校広報の現場では、募集広報や事務作業に追われ、日々の発信や新しい施策まで手が回らないことも少なくありません。Hanaui PRは、学校の強みや伝えるべき価値を整理し、ブランディング設計、広報・PR戦略、発信企画、プレスリリース作成、メディア対応まで、必要なところから一緒に進めます。まずは、今の広報を見渡すところから始めましょう。

Worries

こんなお悩みは、
ありませんか。

専門学校・私立学校の広報で、よく伺うお悩みです。ひとつでも近いものがあれば、お気軽にご相談ください。

  • ✓ 日々の業務に追われ、広報の戦略をじっくり考える時間が取れない
  • ✓ 広報の成果や必要性を、上層部に説明する材料が足りない
  • ✓ 新コース設立やチャンネル追加など、つい「足し算」になりがち
  • ✓ 広報をひとりで抱え込み、学内に相談できる相手がいない
  • ✓ 良い学校なのに、その良さが伝わらない。それが、悔しい
Strength

選ばれる理由

01 On Your Side

学校現場の事情がわかる

教員として12年、学校広報の専任担当として4年。現場の忙しさや校内調整の難しさをふまえ、無理なく進められる広報を提案します。

02 End to End

ブランディングから実務まで相談できる

ブランディング設計、広報・PR戦略、発信企画、SNS・HP、プレスリリース作成、メディア対応まで。必要な支援を分断せず、学校の状況に合わせて組み立てます。

03 Built to Last

担当者が続けられる形にする

外注して終わりではなく、校内で続けられる仕組みづくりまで支援します。月1回の定例から、小さく始めることもできます。

広報・PR戦略・ブランディングについて
Service Flow

ご支援は、三つのステップで。

ブランディング設計から発信の実務まで。学校の状況に合わせて、必要なところから進めます。

STATE 01

今の広報を棚卸しする

現在の広報体制、発信内容、年間スケジュール、課題感を一緒に見渡します。何から手をつけるべきかを整理し、優先順位を決めます。

こんな方に：何から相談すればいいかわからない

STATE 02

方針を決めて、一緒に動かす

ブランディングの方向性や広報・PR戦略をもとに、SNS・HP・プレスリリース・メディア対応などを実際に進めます。

こんな方に：継続的に広報を整えたい

STATE 03

担当者が続けられる形に残す

テンプレート、運用ルール、引き継ぎ資料などを整え、担当者が替わっても広報が止まらない土台をつくります。

こんな方に：将来的には校内で運用したい
